# AB15 Online Compiler 🐍💻

Online code compiler supporting 15+ programming languages with real-time execution.

## Features ✨
- 🌍 **15 Languages**: Python, JavaScript, C, C++, Java, Rust, Go, PHP, Ruby, Swift, Kotlin, TypeScript, Bash, R, C#
- 🎨 **5 Themes**: Dark, Light, Midnight, Forest, Dracula
- ⌨️ **Syntax Highlighting**: Powered by CodeMirror
- ⚡ **Quick Run**: Ctrl + Enter to execute
- 💾 **Auto-Save**: Your code is saved automatically
- 📱 **Responsive Design**: Works on all devices

## Technologies Used 🛠️
- HTML5/CSS3/JavaScript
- CodeMirror Editor
- Piston API for code execution

## Live Demo 🌐
Try it now: [AB15 Compiler](https://abdallah-ab15.github.io/AB15-compiler/)

## How to Use 📖
1. Select your programming language
2. Write your code in the editor
3. Press **Run** or **Ctrl + Enter**
4. See the output in the terminal
